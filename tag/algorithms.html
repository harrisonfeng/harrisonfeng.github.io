Linked List

Mon 10 March 2014 | tags: javaalgorithms

Overview

Definition:

A linked list is a recursive data structure that is either empty(null) or a reference to a node having a generic item and a reference to a linked list.

Algorithms, Fourth Edition

Implementation

Here is singly linked list implementation in Java.

/*
 * Version 1
 */
class Node {
    Node next = null;
    int data;

    public Node(int d) {
        data = d;
    }

    /**
     * Appends the specified element to the end of this list.
     *
     */
    void add(int d) {
        Node end = new Node(d);
        Node n = this;
        while (n.next != null) {
            n = n.next;
        }
        n.next = end;
    }
}

Creat and access linked list

/*
 * Create a linked list
 * 5 -> 3 -> 8
 *
 */
Node head = new Node(5);
head.add(3);
head.add(8);

/*
 * Access node
 */
head.data             // Node 5
head.next.data        // Node 3
head.next.next.data   // Node 8

Simplified

/*
 * Version 2
 */
class Node {
    int data;
    Node next;

    public Node(int d) {
        data = d;
        next = null;
    }
}

Operations

  1. Deleting a Node from a singly linked list
Node delNode(Node head, int nd) {

    /*
     * If the Node to be deleted is head, just move head.
     */
    if(head.data == nd) {
        return head.next; // Move head
    }

    Node n = head;

    while(n.next != null) {
        if (n.next.data = nd) {
            n.next = n.next.next;
            return head; // head didn't changed
        }
        n = n.next;
    }
}
  1. Deleting duplicated Nodes from an unsorted singly linked list
a) Use a buffer

import java.util.Hashtable;

public static void delDupNodes(LinkedListNode n) {

    Hashtable tab = new Hashtable();
    LinkedListNode pre = null; // previous node

    while(n != null) {
        if(tab.containsKey(n.data)) {
            pre.next = n.next;
        } else {
            tab.put(n.data, true);
            pre = n;
        }
        n = n.next;
    }
}
